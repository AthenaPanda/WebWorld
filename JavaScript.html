let命令

用来声明变量，但只在代码块有效。 

            for(let i = 0;i < 10;i++) { 
              ...... 
            }
  • let声明的变量仅在块级作用域内有效.
  • let所声明的变量必须在声明后使用，不存在变量提升.
  • 在代码块内，使用let命令声明变量之前，该变量都是不可用的.let命令声明变量之前，都属于变量的“死区”。
  • 不允许在相同作用域内，重复声明同一个变量。
